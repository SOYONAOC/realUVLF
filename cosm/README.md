# Cosmology Visuals

Run:

```bash
../.venv/bin/python cosm/plot_cosmic_timeline.py
```

This writes:

- `cosm/cosmic_timeline_z0_2000.png`
- `cosm/cosmic_timeline_z0_2000.pdf`

The plot shows `z = 0` to `2000`, cosmic age from `Planck18`, and broad evolutionary stages such as reionization and CMB decoupling.
